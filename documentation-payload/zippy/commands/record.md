## Summary

Record audio - if an input is enabled.
  
- Needs Admin: False  
- Version: 1  
- Author: @ArchiMoebius  

### Arguments

seconds - how long to record

## Usage

```
record seconds
```

## Detailed Summary

The 'record' command will attempt to capture audio data from the primary input - if available.
