## Summary

On Linux and Windows - execute commands using either `bash` or `cmd.exe` - detached process.

- Needs Admin: False
- Version: 1  
- Author: @ArchiMoebius  

### Arguments

Whatever command you want to execute

## Usage

```
spawn command arguments
```

## Detailed Summary
