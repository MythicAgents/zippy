## Summary

Take a screenshot of the specified monitor.
  
- Needs Admin: False  
- Version: 1
- Author: @ArchiMoebius  

### Arguments

monitor - zero indexed.

## Usage

```
screenshot monitor
```

## Detailed Summary

The 'screenshot' command will attempt to capture the users monitor contents as a PNG image.
